OSgenie

More PCs should be Linux Desktops!

Use these ready built tools to showcase and install Linux desktops on any PC.

ISO to PXE server

A collection of scripts to build a server that turns downloaded isos into a PXE menu.

Automatically installs and configures the following:

  1. Transmission BitTorrent Server
  2. Apt-Cacher Server
  3. Squid Caching proxy
  4. PXE Boot Server
  5. DHCP Server (optional)

ISO Updating server

A collection of scripts to build a virtual host that batch updates and modifies LiveCD isos.

Automatically installs and configures the following:

  1. KVM Virtual Server
  2. Bridged network
  3. 64bit and 32bit Virtual Servers configured to update isos
